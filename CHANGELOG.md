# 版本紀錄

本文件遵循 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.1.0/) 的分類方式，產品版本採 [Semantic Versioning](https://semver.org/lang/zh-TW/)。Sites 平台版本與產品語意版本分開記錄。

## [Unreleased]

### Planned

- 自動布局、節點避讓與大型心智圖效能改善。

## [0.10.0] - 2026-07-18

### Changed

- AI 面板改為「選節點、描述卡點、取得三個建議、點擊加入」的單一路徑。
- 建議卡片點擊後直接加入目前節點，不再需要多選與預覽確認。
- 網站由公開存取改為僅限擁有者的私人 Beta。

### Removed

- AI 模式切換、多節點上下文、節點對話歷史與批次建議流程。

### Verified

- 正式建置成功，25 項自動測試通過，並發布至 Sites 平台版本 17。

## [0.9.0] - 2026-07-16

### Added

- 伺服器端 OpenAI Responses API 與結構化建議輸出，API 金鑰僅存在伺服器環境。
- 發散、拆解、質疑三種 AI 思考模式。
- 最多 8 個節點的 AI 上下文選取，以及建議來源節點標示。
- 依地圖與節點保存在本機的 AI 討論紀錄、採用狀態與再次查看功能。
- AI 逾時、配額、金鑰、網路及輸出格式錯誤提示與重試操作。

### Changed

- 既有批次預覽流程改為同時支援真實 AI 建議與離線示範建議。
- 刪除節點或子樹時同步清除對應的本機 AI 討論紀錄。
- PRD 升級至 3.0，完成 M2 真 AI 基礎的 P1-01～P1-05 規格更新。

## [0.8.0] - 2026-07-16

### Changed

- 畫布縮放範圍由 70%–130% 擴大為 50%–200%。
- 左側工具列移除常駐中文小標，改為放大圖示並在滑鼠懸浮或鍵盤聚焦時顯示中文提示。
- 放大搜尋、節點操作、縮放、AI 建議、大綱排序及匯出選單圖示，提升辨識與觸控操作性。
- 手機版工具列寬度同步調整，避免放大後的圖示擁擠。
- PRD 升級至 2.3，記錄新版縮放範圍與圖示可用性規格。

## [0.7.0] - 2026-07-16

### Added

- 心智圖畫布支援滑鼠拖曳空白處平移，以及手機單指觸控平移。
- 大綱模式支援桌面拖放同層節點排序。
- 大綱模式加入手機友善的上移／下移按鈕，排序時保留完整子樹。

### Changed

- 「適合畫面」會同時清除手動平移量，快速找回中心主題。
- 節點拖曳改用位移計算，避免畫布平移後拖曳節點跳動。
- PRD 升級至 2.2，記錄跨裝置平移與結構化大綱排序的需求及驗收條件。

## [0.6.0] - 2026-07-15

### Added

- 畫布與大綱雙模式。
- 節點搜尋、分支收合與適合畫面。
- AI 建議多選、預覽與批次加入。
- 手機版可收合 AI bottom sheet。
- 可編輯文件標題；本機與共享地圖皆會保存。
- Enter、Tab、Delete/Backspace、Esc 等核心鍵盤操作。

### Changed

- 節點編輯由 `window.prompt` 改為內嵌表單。
- 強化選取狀態、觸控目標與手機版操作空間。

## [0.5.0] - 2026-07-15

### Added

- 無登入共享地圖與 `/m/[id]` 路由。
- Sites D1 儲存、版本樂觀鎖與衝突提示。
- 本機草稿自動保存、還原及重設。
- 心智圖純函式、儲存與共享資料測試。

## [0.4.0] - 2026-07-14

### Added

- PNG、PDF、Markdown 匯出。
- 漸細貝茲曲線連接線。
- 節點移除、復原與重做。

## [0.1.0] - 2026-07-13

### Added

- 靈感樹互動式心智圖初版。
- 節點新增、選取、拖曳、縮放與情境式 AI 建議原型。
- 溫暖紙張工作室視覺方向與 Sites 部署。
